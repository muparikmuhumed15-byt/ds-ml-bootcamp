### ***Research Assignment: Introduction to Data Science and Machine Learning***



##### 1\. Definition of Data Science and Machine Learning



Data Science is the process of collecting, cleaning, analyzing, and interpreting data to gain useful insights and support decision-making.



Machine Learning (ML) is a branch of Data Science and Artificial Intelligence that enables computers to learn patterns from data and make predictions without being explicitly programmed.



Relationship Between Them



Machine Learning is a tool within Data Science. Data Science uses ML techniques to build predictive models and solve problems using data.



###### Real-Life Example



Online Shopping (Amazon-like platforms)



Data Science collects customer purchase and browsing data.

Machine Learning analyzes the data and recommends products the customer may like.

Result: Better customer experience and increased sales.



###### 2\. Data Science Lifecycle



###### &#x20;  Stage Description



1\. Problem Definition Identify the business or research problem.

1\. Data Collection Gather relevant data.

1\. Data Cleaning Remove errors and missing values.

1\. Data Exploration Analyze data and find patterns.

1\. Model Building Apply Machine Learning algorithms.

1\. Evaluation Test model performance.

1\. Deployment Use the model in real-world applications.

&#x20;  Where Does Machine Learning Fit?



Machine Learning mainly fits in the Model Building and Evaluation stages because it is used to learn patterns from data and make predictions.



###### 3\. Supervised vs Unsupervised Learning



&#x20;  Feature Supervised Learning Unsupervised Learning



Data Type: Labeled data Unlabeled data

Goal: Predict outcomes Discover patterns

Example: Email spam detection Customer segmentation



Supervised Learning: Predicting house prices using past housing data.

Unsupervised Learning: Grouping customers based on shopping behavior.



###### 4\. Overfitting

&#x20;  What Causes Overfitting?



Overfitting happens when a model learns the training data too closely, including noise and errors, making it perform poorly on new data.



How to Prevent It?



Use more training data.

Remove unnecessary features.

Use cross-validation.

Apply regularization techniques.



###### 5\. Training Data and Test Data Split



The dataset is usually divided into:



Training Data (80%) → Used to train the model.

Test Data (20%) → Used to evaluate the model.



Why Is This Necessary?

It helps check whether the model can perform well on new, unseen data and prevents overfitting.
